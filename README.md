## 🚀 Getting Started

- Clone the repository
- Install dependencies with `npm install`
 ```bash
  npm install
  ```
- Run the development server with `npm run dev`
 ```bash
  npm run dev
  ```
- Open your browser and navigate to `http://localhost:3000`
